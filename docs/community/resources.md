# Resources & Support

Explore official articles, references, and learn how to support the development of AnomotionJS.

---

## 1. Learning Materials & Articles

### [High Performance Text in WebGL](./high-performance-text-webgl)
A detailed guide covering Signed Distance Fields (SDF), Multi-channel Signed Distance Fields (MSDF), glyph instancing, and GPU shaders.

### [Verlet Physics in Web Animations](./verlet-physics-web-animations)
An introductory guide to Verlet Integration numerical solvers, springs math, and interactive attract/repel cursor fields.
