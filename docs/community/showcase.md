# Community Showcase

See what developers around the world are building with AnomotionJS. From immersive landing pages to interactive games, this space is dedicated to exploring real-world projects powered by our high-performance typography engine.

---

## 🚀 Call for Submissions

We are currently building our directory of featured deployments! If you have built a website, application, game, or experiment using AnomotionJS, we want to showcase your work here.

### How to Submit Your Project:
1. **Fork the Repository:** Create a fork of the [AnomotionJS GitHub Repository](https://github.com/EldrexDelosReyesBula/AnomotionJS).
2. **Add Your Details:** Add a new entry to this page (`docs/community/showcase.md`) with the following format:
   - **Project Name:** The name of your site or application (with a link to the live deployment).
   - **Platform/Renderer:** Specify if you used `dom`, `canvas`, `three`, or `webgpu`.
   - **Description:** A short overview of how you used AnomotionJS.
   - **Creator:** Your GitHub username or website.
3. **Submit a Pull Request:** Open a PR targeting the `main` branch. Once reviewed, your project will be featured on this page for all developers to see!
