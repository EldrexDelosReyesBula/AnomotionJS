# AnomotionJS Roadmap

Public direction and timeline of features for AnomotionJS.

## 🎯 Now (Active Development)
* Full WebGPU quad character shader stabilization.
* React, Vue 3, and Svelte 5 package adapters (`@eldrex/anomotionjs-react`, etc.).

## 🚀 Next (Q3 2026)
* Scroll-bound animation triggers.
* SVG Path morphing renderer.

## 💡 Later (Under Consideration)
* Layout shift reconciliation helper.
* Native React Native / Skia renderer backend.
