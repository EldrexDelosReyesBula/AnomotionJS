# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-06-27

### Added
- Created complete WebGPURenderer with Canvas 2D fallback.
- Added synchronous glyph layout initialization to ThreeRenderer.
- Enabled customizable Glitch Color parameter support in core engine.
- Implemented cursor displacement mouse interaction on all renderers.
- Restored Timeline Stagger, Easing, and custom parameter selections in playground.
- Created comprehensive framework integration guides for React, Vue 3, and Svelte 5.

### Fixed
- Fixed unclosed script tags in template variables that broke HTML parsers.
